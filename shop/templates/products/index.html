Descreva o funcionamento do pacote flask_restful

Flask-RESTful é uma extensão para Flask que adiciona suporte para construir APIs REST rapidamente. É uma abstração leve que funciona com seus ORM / bibliotecas existentes. Flask-RESTful incentiva as melhores práticas com configuração mínima.

O Flask 0.7 apresenta visões plugáveis ​​inspiradas nas visões genéricas do Django que são baseadas em classes ao invés de funções. A intenção principal é que você possa substituir partes das implementações e, dessa forma, ter visualizações plugáveis ​​e personalizáveis.


Cite os principais eventos e sinal durante um request, como acessá-los e um exemplo prático.

Evento de pegar os dados na API.

Para fazer a busca de um produto em uma API, podemos usar o verbo GET do protocolo http passando como parâmetro o id do produto na url.

Evento de pegar os dados na API

Para fazer a busca de mais produtos em uma API, podemos usar o verbo POST do protocolo http passando como parâmetro um JSON contendo os campos a ser pesquisado.

Evento de inserir os dados pela API

Para inserir dados através de uma api, podemos usar o verbo POST passando como parâmetro um JSON contendo os campos a ser inserido.

Atualizar dados pela API

Para atualizar dados através de uma API, podemos usar o verbo PUT passando como parâmetro um JSON contendo os campos a ser atualizado.

Deletar dados pela API

Para deletar dados através de uma API, podemos usar o verbo DELETE passando como parâmetro o id direto na url ou por arquivo JSON.
